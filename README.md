# Configurando o Ionic no PC

### 1 - Terminal (CMD)
Abra o seu terminal e navegue até o diretório onde deseja clonar o projeto. Exemplo:
cd\
cd ionic (supondo que esteja no Windows e exista o diretório ionic)

### 2 – Clonando o projeto:
No terminal digite: git clone https://github.com/vidalallan/todo-list-ionic.git

### 3 – Acesse a pasta clonada. Exemplo
cd todo-list-ionic

### 4 – Executando a aplicação
No terminal digite ionic serve; pressione a tecla Enter
Será solicitada a instalação do Angular. Digite Y (sim), e pressione a tecla Enter novamente

### 5 - Visualização do Projeto
Se tudo estiver correto, o navegador será aberto executando a URL: http://localhost:8100/
